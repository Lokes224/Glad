## 🌍 What Glad Is

Glad is a data app for working with shared language files. It helps you manage words, tokens, and model data in one place. It is built for end users who want a simple way to view and use global language data on Windows.

Use Glad if you need a local tool for language sets, shared files, or AI data files. It can help with tasks like browsing content, opening data files, and keeping language resources in one folder.

## 🛠️ Troubleshooting

### The app does not open

- Check that the file finished downloading
- Right-click the app and choose Run as administrator
- Make sure you extracted all files from the zip
- Try opening the app from the folder where it was unpacked

### Windows says it blocked the app

- Right-click the file
- Select Properties
- Look for an Unblock box near the bottom
- If you see it, check it and select Apply
- Open the app again

### Files do not load

- Check that the file is in a supported format
- Make sure the file is not damaged
- Move the file to a simple folder path like `C:\Glad`
- Try a smaller data set first

### Search gives no results

- Check spelling
- Try a shorter term
- Make sure you opened the right folder
- Confirm the words or tokens are in the data set
